# Project: dragoman

PATH şirketinde müşterilerden veya partnerlerden gelen teklif taleplerine istinaden analiz, eforlandırma, teklif hazırlama süreçlerini yürüten web uygulamasıdır.

## Genel İşleyiş

Web uygulaması arayüzünden verilen brife istinaden analiz süreci işletilir. Analiz sürecinin sağlıklı işlemesi için bir questionnaire bulunur. Cevaplara istinaden analiz yapılır.

### Proje Tipleri 
Proje Tipleri analizin nasıl yapılacağına dair temel argümanlardan biridir

- Shopify, 
- Corporate Custom, 
- Tailor-made Custom, 
- Mobile
